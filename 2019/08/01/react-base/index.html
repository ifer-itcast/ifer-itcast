useMemo

使用 function 的形式来声明组件，失去了 shouldCompnentUpdate（在组件更新之前）这个生命周期，也就是说我们没有办法通过组件更新前条件来决定组件是否更新。useMemo 解决此问题，它可以在某些状态变化时才执行某方法。
