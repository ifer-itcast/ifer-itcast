项目介绍

基本内容

项目名称：人力资源管理系统。是企业的后台管理系统，对企业内部信息进行管理。

项目演示：https://heimahr-t.itheima.net/login

项目技术：基于 vue-element-admin 二次开发（不是从 0 开始），使用 ES2015+、vue、vuex、vue-router 、vue-cli 、axios 和 element-ui 等技术。

  • vue-element-admin 官网：https://panjiachen.gitee.io/vue-element-admin-site/zh/

  • vue-element-admin 代码：https://gitee.com/panjiachen/vue-element-admin

  • vue-element-admin 效果：https://panjiachen.gitee.io/vue-element-admin/#/login?redirect=%2Fdashboard

业务难点：权限控制（不同的人进入系统可以看到的菜单是不同的），表单校验，复杂数据处理，excel 导入导出等。

项目运行机制和代码解析

App.vue

分析 Layout 组件的结构布局， src/layout/index.vue 。

permission.js

src/permission.js ：控制页面登录权限的文件，后续会从 0 到 1 重新编写，所以先清空里面的所有代码。

settings.js

src/settings.js ：是项目基本信息的配置文件，里面有三个属性 title（项目名称）、fixedHeader（固定头部）、sidebarLogo（显示左侧菜单 logo）。

port

📝 目标：能够通过环境变量配置项目的端口号。

vue.config.js

  • process.env 是 Node 的语法，可以获取到环境变量。

  • 执行 npm run dev，此时会加载执行 .env.development 文件内容。

  • 执行 npm run build:prod，会加载执行 .env.production 文件内容。

  • 所以，如果想要设置开发环境的接口，直接在 .env.development 中写入对于 port 变量的赋值即可。

🤔 注意：修改服务的配置文件，想要生效的话，必须要重新启动服务，值 8888 后面不能留有空格。

vuex
