Pinia 介绍

  • Pinia 是一个状态管理工具，它和 Vuex 一样为 Vue 应用程序提供共享状态管理能力。

  • Pinia 实现状态管理有两种语法：选项式 API 与 组合式 API，我们学习组合式 API 语法。

  • Pinia 对 TS 的支持很好，具有很好的类型提示。

  • Pinia 可以创建多个全局仓库，不用像 Vuex 一样，一个仓库嵌套模块，结构复杂。

  • 管理数据简单，提供数据和修改数据的逻辑即可，不像 Vuex 需要记忆太多的 API。

  • Pinia 是一个简单实用的状态管理工具，和菠萝一样，香！
