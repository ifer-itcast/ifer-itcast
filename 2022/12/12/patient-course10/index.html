问诊室-websocket介绍

目标

认识 WebSocket。

内容

什么是 WebSocket？

WebSocket 是一种网络通信协议，就类似于 HTTP 也是一种通信协议。

为什么需要 WebSocket?

因为 HTTP 协议有一个缺陷：通信只能由客户端发起。

理解 WebSokect 的通讯过程？

了解 WebSocket API 的含义， index.html 。

我们项目中使用 socket.io-client 来实现客户端代码，它是基于 websocket 的库。

socket.io 是什么？

问诊室-文字聊天

目标

可以发送文字消息，可以接收文字消息。

步骤

  1. 底部操作组件，可以输入文字，触发 send-text 事件传出文字。

  2. 问诊室组件，监听 send-text 事件接收文字。

  3. 获取订单详情，需要使用医生ID，作为收信息人的标识。

  4. 通过 socket.emit 的 sendChatMsg 发送文字给服务器。

  5. 通过 socket.on 的 receiveChatMsg 接收发送成功或者医生发来的消息。

  6. 展示消息。

内容

  1. 输入文字，触发 send-text 事件传出文字，Room/components/RoomAction.vue。
  1. 问诊室组件，监听 send-text 事件接收文字，Room/index.vue。
  1. 通过 socket.emit 的 sendChatMsg 发送文字给服务器。
  1. 通过 socket.on 的 receiveChatMsg 接收发送成功或者医生发来的消息。
  1. 展示消息，Room/components/RoomMessage.vue。
