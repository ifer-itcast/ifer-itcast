动态主题的设置

目标

实现动态主题的设置。

内容

我们想要实现在页面中实时的切换颜色，此时页面的主题可以跟着设置的颜色进行变化。

ElementUI 自带切换效果

  1. 颜色选择器组件，ElementUI 自定义主题方案。

  2. 新建 styles/element-variables.scss 文件。

  1. main.js 中导入。
  1. 点击【员工管理】观察按钮颜色。

问题：主题色写死了。

解决：根据我们选择的颜色动态配置主题色，直接使用 vue-element-admin 封装好的组件即可。

使用 ThemePicker 组件

官方演示地址

  1. 拷贝 02-笔记 => 资源 => 组件 => ThemePicker 到项目的 components 目录，源代码地址。

  2. 注册组件。

  1. 把 theme-picker 组件放置于 layout/components/Navbar.vue 中。
  1. 注释掉 main.js 中引入的样式。

定义默认主题色

需求：已进入页面，希望默认就是绿色怎么做呢？

  1. 分析 components/ThemePicker/index.vue 中的代码。

  2. 准备默认主题色 theme 属性（注意格式: #xxxxxx），store/modules/settings.js。

  1. 改造 watch 代码, 添加 immediate: true，components/ThemePicker/index.vue。

持久化主题

  1. 注册 change 事件监听，layout/components/Navbar.vue。
  1. 准备 action，store/modules/settings.js

退出时清除主题色信息

  1. 提供 mutation resetTheme，store/modules/settings.js
  1. 退出的 action 操作，store/modules/user.js

通过行内样式自定义自己的主题

  1. 取出 store 中的 theme，通过行内样式应用到侧边栏，layout/components/Sidebar/index.vue。
  1. 应用到 Navbar，layout/components/Navbar.vue。

通过切换类名自定义样式（体力活）

  1. 通过计算属性，根据选中的颜色（store 中的 theme）计算出 classObj，例如颜色是 #6959CD，计算出的 classObj 就是 { 'purple-theme': true }，App.vue。
  1. 准备多套主题样式，通过计算出的 class 作用于页面中的其他模块，styles/theme.scss。
  1. 引入主题样式文件，styles/index.scss。

实现多语言功能

目标

实现国际化语言切换。

初始化多语言包

本项目使用国际化 i18n 方案，通过 vue-i18n 实现。

  1. 安装国际化的包。
  1. 创建多语言实例，lang/index.js。
  1. 挂载 i18n 插件到 Vue 项目，main.js。
  1. 点击组织架构页面的删除按钮进行观察。

引入自定义语言包

此时，ElementUI 确实变成了 zh/en（中文/引文），但是我们常规的内容怎么进行切换呢？

  1. 复制 02-笔记 => 资源 => 语言包 中的文件到项目的 lang 目录。

  2. 引入该语言包，lang/index.js。

  1. 在左侧菜单应用自定义语言包，layout/components/Sidebar/SidebarItem.vue。

当我们全局注册 i18n 的时候，$t 方法会被挂载到 Vue 原型上，每个组件都可以直接使用，这个方法会根据传入的 key，自动的去寻找当前语言的文本。

  1. 我们可以自己添加一些中英翻译，将头部翻译成中英文。

集成多页签组件

目标

实现 tab 页打开路由的功能。

内容

可以引入多页签组件，使页面显示更加有效率，多页签的组件的代码过于繁杂，开发实际需要的是集成和调用能力，所以我们这儿只是将开发好的组件集成到当前的功能项中即可。

  1. 把 02-笔记 => 资源 => 多页签 => TagsView 组件放置到项目的 components 目录，并在 components/index.js 中全局注册。
  1. 将 02-笔记 => 资源 => 多页签 => vuex => tagsView.js 放置到项目的 store/modules 目录，并在 store/index.js 中引入该模块。
  1. 引入 tagsView 组件，layout/index.vue。
