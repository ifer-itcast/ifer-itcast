原生 svg-sprites 的原理

基本使用

上面使用使用了 svg-icon 组件展示 svg 图标，内部使用了 svg-sprites 技术。

svg-sprites 原理：利用 svg 的 symbol 元素，将每个 icon 包括在 symbol 中，通过 use 元素使用对应的 symbol。

svg-sprites 使用参考：懒人神器：svg-sprite-loader 实现自己的 Icon 组件，未来必热：SVG Sprites 技术介绍。

原生使用步骤如下。

  1. 把 svg 替换成 symbol 元件，加上对应的 id，然后外部用一个大的 svg 包括（元件的特点是不会显示在页面中，但可以包裹 svg icon）。
  1. 页面中任何位置，通过 use，根据 id 显示图标。

进阶使用

上面写法的两个问题：每次自己去准备 symbol 元件麻烦；每次自己去写 svg use 麻烦。

  1. svg-sprite-loader 可以把通过 require.context 动态引入的所有文件塞到一个个 symbol 中，symbol 的 id 如果不特别指定，就是文件名。

  2. 通过封装组件，控制 use xlink:href 对应的值。
