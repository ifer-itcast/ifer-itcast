图片懒加载

介绍

  • 电商类网站，图片会非常的多，而如果一上来就加载所有的图片，会导致网页加载很慢。

  • 图片懒加载：等图片正式进入到可视区中时，才加载对应的图片，否则不请求图片。

  • 如何知道图片进入或者离开了可视区？useIntersectionObserver。

演示

useIntersectionObserver 的基本使用，src/views/playground/index.vue。

封装

可以封装指令 v-lazy 实现图片懒加载，directives/index.ts。

数据懒加载

概述

  • 电商类网站，尤其是首页，内容有好几屏，而如果一上来就请求并渲染所有屏的数据，会导致首页加载很慢。

  • 数据懒加载：等组件正式进入到可视区中时，才把组件内部的 AJAX 请求发起，否则不请求数据。

  • 优化人气推荐。

封装

  • src/utils/hooks.ts。
  • 优化新鲜好物和人气推荐。

Vue 动画

基本介绍

在 Vue 中，显示隐藏/创建销毁一个元素或组件的时候，可以通过内置的 transition 组件实现动画。

Vue2 的动画类名

Vue3 的动画类名

  • 进入（显示，创建）：v-enter-from 进入前；v-enter-active 进入中；v-enter-to 进入后。

  • 离开（隐藏，销毁）：v-leave-from 进入前；v-leave-active 进入中；v-leave-to 进入后。

🧐 多个 transition 使用不同动画，可以添加 name 属性，name 属性的值替换掉 v 即可。

滚动加载商品的 bug

  • 产品区域需要滚动比较多时才去加载数据。
